# Medical-System (C++ Terminal)

## Requirements
To compile the provided source code, your compiler must have support for at least C++ 11.

## Features
- Patient Management
- Administrative Functions
- CSV Data Handling

## How to install
1. Clone the Repository 
2. Open a Terminal
3. Compile (g++ main.cpp second.cpp -o main)
4. Run (./main)
5. Follow Prompts
